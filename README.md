# 🌾 Farmer Credit Score — விவசாயி கடன் மதிப்பீடு

A multilingual web app that helps Tamil Nadu farmers calculate their credit score and find nearby banks.

## 🚀 Features
- Credit Score Prediction using Machine Learning
- Multilingual Support — Tamil, Hindi, English
- Nearby Bank Finder with interactive map
- EMI Calculator for loan planning
- Government Schemes info (KCC, PM-KISAN, PMFBY)

## 🛠️ Tech Stack
- Python + Streamlit
- Scikit-learn (ML model)
- Folium (interactive map)
- Pandas / NumPy

## 📦 Installation
pip install -r requirements.txt
streamlit run app.py

## 📁 Files
- app.py — Main application
- score_model.pkl — Trained ML model
- scaler.pkl — Feature scaler
- requirements.txt — Dependencies
- dataset.csv — Training dataset
